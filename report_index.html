研究結論

各研究問題之統計分析結果摘要

Q1 差異顯著

公立 vs 私立 — 休學率差異

0.090

公立院校的平均休學率（0.090）顯著高於私立院校（0.077）。Welch t 檢定結果 p = 0.003，差異在統計上具有顯著意義（α = 0.05）。

可能原因：公立院校入學門檻較高，學生在遭遇課業壓力或生涯轉換時，傾向以休學作為暫緩選擇；而私立院校學費較高，學生可能更傾向持續就讀以回收教育投資，但在面臨困境時也可能因此承受更大壓力。

Q2 方向相反，無正向關聯

生師比 — 與休學率的相關性

r = -0.205
Pearson 相關係數
p = 0.035
顯著水準
n = 106
學校樣本數

生師比與休學率之間的 Pearson 相關係數為 r = -0.205（p = 0.035），方向與理論預期相反（生師比越高，休學率反而越低）。統計上的顯著性可能來自樣本特性，不宜解讀為生師比對休學率有實質正向影響。

理論預期生師比高時教學資源稀缺，學生壓力增加，休學率應上升；然而實際資料呈負相關，顯示生師比與休學率之間並無實質正向關聯。影響休學決策的因素可能更多來自個人身心狀況、經濟壓力或生涯規劃，與師資配置關係有限。

單因子 ANOVA 顯示各地理區域間的休學率存在顯著差異（F = 2.479, p = 0.044）。休學率最高為東部（0.098），最低為中部（0.072）。

可能原因：東部院校數量少、外縣市學生比例高，交通不便與生活適應困難可能提高休學意願；中部地區院校多位於都會周邊，生活機能完善，學生就讀穩定性相對較高。
